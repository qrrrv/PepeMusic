## 🚀 Как собрать APK (2 способа):

### СПОСОБ 1: Через Android Studio (РЕКОМЕНДУЕТСЯ) 🌟

1. **Открой Android Studio**

2. **File → Open** → Выбери папку `SimpleMusicPlayer`

3. **Дождись синхронизации Gradle**
   - Внизу появится "Gradle sync in progress..."
   - Подожди пока закончится (2-5 минут)

4. **Build → Build Bundle(s) / APK(s) → Build APK(s)**
   - Или нажми Ctrl+B (Windows) / Cmd+B (Mac)

5. **Дождись окончания сборки**
   - Внизу появится "Build successful"
   - Появится уведомление "locate" - нажми на него

6. **APK готов!**
   - Файл находится в: `app/build/outputs/apk/debug/app-debug.apk`
   - Или: `app/build/outputs/apk/release/app-release.apk`

### СПОСОБ 2: Через командную строку ⚡

**Windows:**
```cmd
cd SimpleMusicPlayer
gradlew assembleDebug
```

**Linux/Mac:**
```bash
cd SimpleMusicPlayer
./gradlew assembleDebug
```

APK будет здесь: `app/build/outputs/apk/debug/app-debug.apk`

## 📋 Debug vs Release APK

### Debug APK (для тестирования):
```bash
gradlew assembleDebug
```
✅ Быстрая сборка (2-3 минуты)
✅ Можно сразу устанавливать
❌ Больше размер (~10-15 MB)
❌ Медленнее работает

### Release APK (для публикации):
```bash
gradlew assembleRelease
```
✅ Оптимизированный
✅ Меньше размер (~5-8 MB)
✅ Быстрее работает
❌ Дольше собирается (5-7 минут)

## 📱 Как установить на телефон:

1. **Скопируй APK** на телефон (через USB или облако)

2. **На телефоне:**
   - Открой файл APK
   - Разреши установку из неизвестных источников (если спросит)
   - Нажми "Установить"

3. **Готово!** 🎉
   - Приложение появится в списке приложений
   - Иконка: стандартная Android иконка

## 🐛 Если что-то не работает:

### Ошибка: "SDK location not found"
**Решение:** Создай файл `local.properties` в корне проекта:
```
sdk.dir=C:\\Users\\ТвоёИмя\\AppData\\Local\\Android\\Sdk
```

### Ошибка: "Gradle sync failed"
**Решение:**
1. File → Invalidate Caches → Invalidate and Restart
2. Дождись перезапуска Android Studio
3. Попробуй снова

### Ошибка: "gradlew not found"
**Решение (Windows):**
```cmd
.\gradlew assembleDebug
```
(с точкой и слэшем в начале!)

### APK не устанавливается на телефон
**Решение:**
- Разреши установку из неизвестных источников
- Проверь: Android 7.0 или выше
- Освободи место (~50 MB минимум)

## 💡 Полезные советы:

### Как найти APK после сборки:
В Android Studio внизу появится:
```
BUILD SUCCESSFUL in 2m 15s
locate or analyze the APK
```
Нажми "locate" - откроется папка с APK!

### Как изменить название приложения:
Открой: `app/src/main/res/values/strings.xml`
Измени:
```xml
<string name="app_name">Твоё Название</string>
```

### Как изменить цвета:
Открой: `app/src/main/res/values/colors.xml`
Измени цвета как хочешь!
